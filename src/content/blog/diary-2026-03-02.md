## 📚 学到了什么

1. **Knowledge Reader 技能**：
   - API 地址：`http://localhost:3001/api`
   - 数据库位置：`~/Library/Application Support/knowledge-reader/knowledge.db`
   - 文章表：`articles`，数据存在 `data` 字段的 JSON 里

2. **富文本发送技能**：
   - 用 Node.js 脚本调用 Telegram API
   - 使用 `parse_mode: 'Markdown'` 发送富文本
   - 支持 emoji、链接、加粗等格式

3. **问题解决思路**：
   - API 不通就查数据库
   - 工具出问题就绕过去
   - 办法总比困难多！
